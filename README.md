# Títulos e descrições via IA.

Selecione um vídeo para que ia gere um título ou uma descrição com base no vídeo selecionado.

<h3>Iniciar projeto local: </h3>
<p>instalando pacotes</p>

```bash
npm install
```
<br>
<p>Executando aplicação</p>

```bash
npm run dev
```
Acesso
```bash
http://localhost:5173/
```
